## ❗ Troubleshooting

| ❌ Problem | ✅ Fix |
|---|---|
| Save won't load | Ensure it's a valid .sts2 file, check file isn't locked by the game |
| "Corrupted save" error | Restore from backups/ folder, re-edit from clean save |
| Cards not appearing in-game | Card ID may have changed — update to latest editor version |
| Gold resets after loading | Apply edits while game is closed, not during a run |
| Floor skip crashes game | Some floor transitions require specific event flags — skip to act start instead |
| Seed not matching | Seeds are version-specific — ensure game version matches seed database |
